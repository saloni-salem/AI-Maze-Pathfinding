AI Maze Pathfinding Game

This is a small AI project made using Python and Flask.

In this project, the computer finds the shortest path in a maze using the A* algorithm.
The maze is generated randomly and the AI moves step by step to reach the treasure.

Technologies used:
Python (Flask)
HTML
CSS
JavaScript

How to run the project:
1. Download the project
2. Open the folder in VS Code
3. Run app.py
4. Open the link shown in the terminal

This project is made for learning basic Artificial Intelligence concepts.
